## 🚀 使用方法

### 快速启动

```bash
# 进入项目目录
cd /Users/wangbo/.openclaw/workspace/projects/paper-research-assistant

# 一键启动（自动创建虚拟环境、安装依赖、启动服务）
./start.sh

# 或者手动启动
source venv/bin/activate
python api_server.py

# 访问Web界面
open http://localhost:8088

# 停止服务
./stop.sh
```

### 命令行使用

```bash
# 激活虚拟环境
source venv/bin/activate

# 运行研究
python run.py "Transformer" --max-papers 30

# 运行测试
python test_features.py
```

### API接口

所有API接口已完整实现并测试通过：

- `GET /api/status` - 获取系统状态
- `POST /api/research` - 开始研究
- `GET /api/papers` - 获取论文列表
- `POST /api/qa` - 智能问答
- `GET /api/qa/history` - 获取问答历史
- `POST /api/qa/clear` - 清空问答历史
- `POST /api/report/generate` - 生成报告
- `GET /api/report/list` - 列出报告
- `POST /api/data/save` - 保存数据
- `GET /api/data/load` - 加载数据
- `GET /api/data/list` - 列出保存的数据
- `POST /api/data/export` - 导出数据
- `POST /api/data/import` - 导入数据
- `POST /api/cache/clear` - 清除缓存
- `POST /api/feishu/bitable` - 创建多维表格
- `POST /api/feishu/wiki` - 归档知识库
- `POST /api/feishu/push` - 发送推送
