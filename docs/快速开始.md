# AI论文全自动预研助手 - 快速开始

## 📦 安装

### 1. 创建虚拟环境
```bash
cd /Users/wangbo/.openclaw/workspace/projects/paper-research-assistant
python3 -m venv venv
source venv/bin/activate
```

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 配置
编辑 `config/config.yaml` 配置文件。

## 🚀 使用方法

### 基本使用
```bash
python3 scripts/main.py --topic "大模型在K12英语口语教学中的应用"
```

### 输出文件
- `output/research_report.md` - 研究报告（Markdown格式）
- `output/papers_comparison.xlsx` - 论文对比表格（Excel格式）

## 📊 功能演示

### 1. 智能文献检索
- 自动检索arXiv、Google Scholar、知网
- 智能去重和相关性排序
- 生成论文元数据表格

### 2. 深度阅读与提取
- 自动解析PDF
- 提取核心观点、实验数据、结论
- 生成对比表格

### 3. 自动生成综述
- 基于LLM自动撰写综述草稿
- 自动添加引用格式
- 生成文献地图

### 4. 每日追踪与推送
- 定时监测新论文
- 突破性进展自动推送
- Feishu实时通知

## 📝 示例输出

运行后，系统会自动生成：

1. **研究报告**：Markdown格式，包含完整的研究信息、文献综述、参考文献等
2. **论文对比表格**：Excel格式，包含所有论文的元数据对比

## 🎯 实际效果

- **效率提升：** 将一周的文献调研工作压缩到30分钟
- **时间节省：** 7天工作 → 30分钟完成
- **信息全面：** 多源检索，不遗漏重要论文
- **知识管理：** 自动构建结构化的本地知识库

## 🔧 配置说明

### 数据源配置
```yaml
data_sources:
  arxiv:
    enabled: true
    max_results: 50
  scholar:
    enabled: false
  cnki:
    enabled: false
```

### 检索配置
```yaml
search:
  keywords: []
  exclude_keywords: []
  date_range: "last_3_years"
```

### 推送配置
```yaml
notification:
  enabled: true
  platform: "feishu"
  daily_check_time: "08:00"
```
