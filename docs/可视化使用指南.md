## 📱 功能说明

### 搜索功能
- **输入关键词**：支持中英文，例如"machine learning"、"artificial intelligence"
- **搜索按钮**：点击搜索按钮开始检索
- **自动刷新**：页面加载时会自动搜索示例数据

### 统计信息
- **论文总数**：找到的论文数量
- **独特作者**：所有论文的作者去重后的数量
- **研究领域**：所有论文的分类去重后的数量
- **检索耗时**：从搜索到显示结果的总时间

### 数据可视化

#### 1. 论文发表趋势图
- **内容**：显示最近10篇论文的发表时间
- **类型**：折线图
- **用途**：了解论文发表的时间分布

#### 2. 论文分类分布图
- **内容**：显示论文最多的Top 10分类
- **类型**：柱状图
- **用途**：了解研究领域的分布情况

### 论文列表

每篇论文显示：
- 📝 **标题**：论文完整标题
- 👤 **作者**：所有作者姓名
- 📅 **发布时间**：论文发表日期
- 🏷️ **分类**：论文所属领域
- 📄 **分类数量**：论文涉及的分类数
- 📥 **PDF下载**：直接下载论文PDF

## 🔧 自定义配置

### 修改默认搜索词

编辑 `index.html` 文件，找到以下代码：

```javascript
window.onload = function() {
    document.getElementById('searchInput').value = 'machine learning';
    searchPapers();
};
```

修改 `'machine learning'` 为你想要的关键词。

### 修改最大结果数

编辑 `index.html` 文件，找到以下代码：

```javascript
const response = await fetch(
    `https://export.arxiv.org/api/query?search_query=allintitle:"${encodeURIComponent(query)}"&start=0&max_results=50&sortBy=submittedDate&sortOrder=descending`
);
```

修改 `max_results=50` 为你想要的数量（1-2000）。

### 修改图表显示数量

编辑 `index.html` 文件中的 `generateCharts` 函数：

```javascript
// 修改趋势图显示数量
const recentPapers = papers.slice(0, 10);  // 改为 5, 20, 50 等

// 修改分类图显示数量
const sortedCategories = Object.entries(categoryCounts)
    .sort((a, b) => b[1] - a[1])
    .slice(0, 10);  // 改为 5, 15, 20 等
```
